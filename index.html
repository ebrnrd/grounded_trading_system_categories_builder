How to use

1. Load files
Start by loading your grounded_trade_categories.ltx. Then load as many .ltx config files (like w_akm.ltx, items_medical.ltx) and .xml string files (like st_items_weapons.xml) as you want. The tool cross-references them automatically to show real item names and descriptions.

2. Browse items
The left panel shows all known item IDs. Hover any item for its full name and description. Filter by "Unplaced" to find items not yet in any section. Use the category chips to filter by prefix group.

3. Edit sections
The center panel shows all sections. Drag items from the left panel into a section. Drag items between sections. Use the + Item field in each section footer to add manually. Click ✕ to remove. Sections support inheritance (the [name]:parent syntax).

4. Create sections
Type a name and optional parent in the toolbar at the top of the center panel, then click + Section. Parent names are autocompleted from existing sections.

5. Export
Click Refresh Export in the header to regenerate the .ltx output. Download or copy it.
